## Project Description
This project is part of the Open Source Software course. It includes analysis of Python as open-source software and 5 Linux shell scripts demonstrating system and scripting concepts.

## Scripts Description

### Script 1: System Identity Report
Displays system information like OS, kernel, user, uptime, and date.

### Script 2: FOSS Package Inspector
Checks if Python is installed and shows version and details.

### Script 3: Disk and Permission Auditor
Displays size and permissions of important system directories.

### Script 4: Log File Analyzer
Counts occurrences of a keyword (like error) in a log file.

### Script 5: Open Source Manifesto Generator
Generates a personalized open-source philosophy statement.
